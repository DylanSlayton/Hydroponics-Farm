This project is a DIY attempt at a hydroponic tower garden and has two parts; the script and the CAD files

Script: Dictates pump operation, generates a website featuring configurable pump operation duration, manual override, and pump status

CAD Files and Structure Methodology: Modular 4 piece modules for different segments, bottom, middle, top, and the hose adapter. Each 90 degree module has dovetails that connect each other to form a full circle that is the same diameter as a standard 5 gallon bucket (should be food grade) that is painted to block sunlight to prevent algae growth. Bucket adapter pieces fit along the lip of the bucket. Standard modules can be stacked infinitely on themselves and of course on bucket pieces. 4 of the lid and top pieces dictates the end of tower, where the nutrient solution is distributed.
